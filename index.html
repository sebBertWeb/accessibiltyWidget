Accessibility Options Widget Project

Paragraph explaining why accessibility is essential for websites

How it works

Using a single javascript file, the accessibility widget is created on any page where it is added. Basic style choices like colour and size are passed as arguments, making the widget versatile and very easily transferable between website. Through the use of local storage, user accessibility preferences are saved and displayed when the user switches pages.

Functionalities

  • Dyslexia friendly font
  • Text enlarging
  • Reading guide (Devices with cursors only)
  • High contrast
  • Link Highlighting
